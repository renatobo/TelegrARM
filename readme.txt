== Description ==
For select ARmember user events, send a notification to a Telegram bot.
This is more of a framework than a fully developed plugin.

== Installation ==
Have ARmember installed and configured 
Create a bot on Telegram following https://core.telegram.org/bots/tutorial#introduction
